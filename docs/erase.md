## erase(object, key) â‡’ <code>boolean</code>
> Deletes a property from an object, if the property exists.


| Param | Type |
| --- | --- |
| object | <code>Object</code> | 
| key | <code>String</code> | 

**Example**  
``` javascript
import { erase } from 'object-agent';

const foo = {
    bar: 1,
    also: 2
};

erase(foo, 'bar');

console.log(foo);
// => {also: 2}
```
